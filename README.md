## Specifikáció

### Bevezetés

1. Ötlet

    Egy egyszerű Android alkalmazás, amely kriptovaluták árfolyamairól tud információkat adni.

2. Cél

    - Legalább 2 képernyő (listázó + részletek)
    - API hívás
    - Helyi DB
    - ORM
    
3. Platform
    
    - Natív Android
    - Min API 21
    
4. Design

    Justinmind wireframe ábrák (content + lo-fi)
    Nincs előre megadott design
    
5. Határidő

    2020.05.11
